Detailed Description

template<uint8_t Size>
class PortExpander74HC595< Size >

This class describes a shift register 74HC595 as an expander: a circuit which provides 8 pins per circuit, and which can be linked to another circuit to increase the number of output pins whitout increasing the number of pins of the Arduino used ! Each expander is identified by a number : the id. Only output pins are used in Accessories. This expander can only provide digital output.
